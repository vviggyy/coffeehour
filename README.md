# coffeehour
*how long do you have to wait in line for free coffee?*

Everyday, the Postgrads (& others) wait in line for free coffee on the 11th floor of the Wu Tsai Institute, but, often, wait times are 15+ plus. Can we predict the wait time from only a few variables?
* number of people ahead of you
* time of arrival
* number of milk + powder refills (probabilitistic)
